# Revive Medical Center Project

A web application designed to simplify appointment and service management for a medical center.

## Features
- Appointment booking system
- Patient and service management
- Organized data handling
- Clean and user-friendly structure

## Technologies Used
- React.Js
- Node.js
- MongoDB
- REST APIs

## Project Overview
This project was created to improve how patients and medical centers manage appointments and services online. It focuses on structured backend handling and clear user interaction.
